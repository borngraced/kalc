### kalc


math kalculator in cli 

```
cargo install kalc
```

#### Usage
Basic operations:
`kalc 2 + 3 * 4`
Using parentheses:
`kalc 5 + 3 / 2`
Help and options:
`kalc --help`
```
```

#### Build from source

```
git clone https://github.com/yourusername/kalc
cd kalc
cargo build --release
```
